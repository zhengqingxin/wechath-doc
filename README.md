# www.thinkjs.org

ThinkJS website code

![www.thinkjs.org](http://p0.qhimg.com/d/inn/20b8a871/1.jpg)

## Installation

```
git clone git@github.com:75team/www.thinkjs.org.git
```

## Install dependencies

```
npm install
```

## Compile

```
npm run compile
```

## Start server

```
npm start
```

## How to visit

Open url `http://127.0.0.1:7777/` in browser.
